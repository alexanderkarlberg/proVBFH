#+TITLE: Qmin study
#+STARTUP: logdone entitiespretty latexpreview

This folder contains a short study of the impact of the cutoff value Qmin

*Parameters*
 - itmx = 6
 - ncall = 500000
 - xmur = 1.0
 - xmuf = 4.0
 - pdf = MSTW2008nnlo
 - scale choice = 1
 
The default value of Qmin, obtained from the PDF, is typically 
 : Qmin = 1 GeV

The conclusion is that multiplying the alphas factors at the
evaluation of the structure function in the matrix element is more
stable, but both implementations give the same answer when
implementing the cutoff correctly.

* Incorrect treatment of Qmin in H+2j phase space (pre [2015-05-06])

In the previous version of the phase space generation, there was no
Qmin cutoff imposed on the momentum of the W/Z scatterer.
