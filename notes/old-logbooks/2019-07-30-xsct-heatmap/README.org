# Heatmap plots

Heatmap plots for comparisons of fact and non-fact NNLO cross section

Requires output of the heat_map.f analysis, and a replacement

  sed -i 's/D/E/g' pwg-NNLO.top

on the output files (here named total_distrib_formatted.top)

To produce a figure, run

  python heatmap.py total_distrib_formatted.top --output heatmap.pdf
